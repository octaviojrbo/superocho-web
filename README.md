# SuperOcho.com.mx

Primera versión de la tienda Super 8.

Incluye:
- Página principal
- Catálogo
- Carrito
- Panel administrador local
- Footer con Inicio, Nosotros, Sucursales, Facebook, Instagram y WhatsApp
- Espacio preparado para Mercado Pago
- Espacio preparado para Estafeta

IMPORTANTE:
Esta versión funciona como base visual y funcional. Para pagos reales con Mercado Pago y guías automáticas de Estafeta se necesita backend y credenciales oficiales.
